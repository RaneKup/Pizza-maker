проект Pizza maker - консольный проект на языке программирования python
Создан учеником Зайцев Егор группа ии-72
Проект представляет собой консольную программу с возможностью создания аккаунтов или входа в них, в проекте можно заказать уже готовую пиццу или напитки а так же есть кастомная пицца в которой можно выбрать ингредиенты самому 
аккаунты хранятся в базе данных json, так что даже после завершения кода и при повторном запуске в аккаунт можно будет заново войти
